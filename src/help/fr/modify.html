Altérer la base de Données


Si vous avez besoin de modifier une entrée déjà existante, vous pouvez effectuer les actions suivantes:

  • Double cliquez sur la cellule que vous désirez éditer
  • Séléctionnez la cellule que vous désirez éditer, puis appuyez sur le bouton "Éditer"


Une boite de dialogue apparaitra vous permettant de modifier le contenu de la cellule. Si vous désirez éditer un mot de passe, vous verrez également un générateur, dont le fonctionnement est identique à celle vue précédemment lors de la création de nouveaux identifiants. Veuillez vous référer à la page Créer une Nouvelle Entrée pour de plus amples informations concernant le fonctionnement du générateur.

Le bouton "Effacer", sans surprise, effacera la ligne séléctionnée de la base de données.
