Créer une Nouvelle Entrée


Pour créer un nouveau set d'identifiants, appuyez sur le bouton "Nouveau". Il vous sera demandé de remplir les champs suivants, mais vous pouvez très bien les laisser vides:

  • Nom d'utilisateur: Le nom d'utilisateur du set d'identifiants sur le point d'êonflextre crée. Par exemple une addresse éléctronique, un pseudonyme...
  • Raison: Le contexte d'utilisation du set d'identifiants. Par exemple un site web, ou une application.
  • Mot de passe: Le mot de passe du set d'identifiants.


Bien que vous soyez libres d'entrer un mot de passe de votre propre création, vous pouvez également utiliser le système de génération de mots de passe, si vous le désirez.
Le générateur utilise les mutateurs suivants:
  • Longueur: Spécifie la longueur désirée du mot de passe généré, en nombre de caractères..
  • Majuscules: Affecte l'inclusion de majuscules dans le mot de passe généré.
  • Chiffres: Affecte l'inclusion de chiffres dans le mot de passe généré.
  • Caractères spéciaux: Affecte l'inclusion de caractères non-alphanumériques dans le mot de passe généré.


Une fois les champs remplis selon vos souhaits, appuyez sur "Valider" pour créer une entrée dans la base de données.
