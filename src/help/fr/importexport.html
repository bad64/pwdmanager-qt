Importer et Exporter


Dans le menu "Fichier", vous trouverez des fonctions ayant pour but de manipuler la base de données comme un unique objet, plutôt qu'une collection de données:

La fonction "Exporter" sauvegardera le contenu de la base de données dans un fichier avec l'extension .crdb à l'emplacement de votre spécification. Vous pourrez ensuite par exemple la transférer vers une autre machine, la sauvegarder dans le Cloud de votre choix, etc.

La fonction "Importer" vous permettra d'importer une base de données exportée précédemment, ou une copie de sauvegarde (habituellement un fichier portant l'extension .crdb.bak). Cependant, vous pouvez également l'utiliser pour importer une base de données utilisant l'ancien format (< 2.1.0, portant l'extension .db ou .db.bak). Le programme convertira automatiquement le fichier vers le nouveau format dans ce cas.

La fonction "Fusionner" ajoutera les entrées d'un fichier préalablement exporté à celles de la base de données actuelle, en éliminant les doublons autant que possible.
