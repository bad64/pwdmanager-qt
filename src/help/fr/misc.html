Autres Opérations


En plus des opérations décrites dans les autres sections de ce manuel, vous avez également accès aux opérations suivantes:

  • Séléctionnez une cellule, puis cliquez sur "Copier" pour copier le mot de passe de la ligne correspondante dans votre presse papiers.
  • Séléctionnez une cellule, puis appuyez sur Ctrl+Haut pour déplacer la ligne vers le haut, ou Ctrl+Bas pour la déplacer vers le bas.
  • Vous pouvez instantanément cacher la base de données en appuyant sur Ctrl+H. Utile si quelqu'un de mal intentionné regarde par dessus votre épaule ! Appuyez sur Ctrl+H pour rendre la base à nouveau visible.
