Creating a New Entry


To create a new set of credentials, press the "New Entry" button. You will be asked to fill the following fields, although you can choose to leave any of them blank:

  • Username: The username for the credentials pair you are about to create. Can be an e-mail address, a screen name, etc...
  • Purpose: What you will use the credentials for, i.e. a website, or application.
  • Password: The password for the credentials pair you are about to create.


Although you can input a password of your own creation, you can also use the integrated password generation facility if you desire.
The password generator uses the following mutators:
  • Length: Specifies the desired length of the generated password, in number of characters.
  • Uppercase characters: Instructs the generator to include uppercase letters.
  • Numeric characters: Instructs the generator to include numbers.
  • Special characters: Instructs the generator to include non-alphanumeric characters.


Once the fields are filled to your liking, press the "Validate" button to write a new entry to your database.
