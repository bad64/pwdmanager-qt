Altering the Database


Should you be in need of modifying a previous entry, you can perform either of the following actions:

  • Double clicking on the cell you wish to edit
  • Highlight the cell you wish to edit, then press the "Edit" button


A dialog box will pop up, prompting you to enter new information. If the field you wish to edit is the password field, you will also be presented with a password generation box, identical in function to the one you were presented with during the initial credentials creation. Please refer to the Create a New Entry page for further information about password generation.

The "Delete" button will, predictably, erase the entire row from the database.
